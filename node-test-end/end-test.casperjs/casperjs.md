# casperjs

## install PhantomJS
```
npm install phantomjs-prebuilt -g
phantomjs.cmd --version
```

## install python]

```
python --version
```

## install CasperJS
```
$ npm install casperjs -g
$ casperjs.cmd --version
```

```
$ node ./src/app.js

$ node ./src/app2.js

$ node ./src/screenshots.js
```
